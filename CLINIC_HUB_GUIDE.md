# 🦷 คู่มือการตั้งค่าระบบศูนย์กลางคลินิก (CIKI Clinic Hub Guide)

คู่มือนี้จะช่วยให้คุณตั้งค่าเครื่องคอมพิวเตอร์หลัก (Master PC) เพื่อให้เครื่องอื่นๆ ในคลินิก (เช่น iPad) เข้าใช้งานได้ผ่าน Wi-Fi และเปิดให้ลูกค้าจองผ่าน LINE ได้จากอินเทอร์เน็ต

---

## 1. การเตรียมเครื่องแม่ (Master PC)

เครื่องแม่ควรเป็นเครื่องที่เสถียรและเปิดทิ้งไว้ตลอดเวลาทำการ

### วิธีหาเลข IP ของเครื่องแม่:
1. กดปุ่ม `Windows + R` บนคีย์บอร์ด พิมพ์ `cmd` แล้วกด Enter
2. พิมพ์คำสั่ง `ipconfig` แล้วกด Enter
3. มองหาบรรทัดที่เขียนว่า **IPv4 Address** (ตัวอย่างเช่น `192.168.1.50`)
4. **จดเลขนี้ไว้** นี่คือที่อยู่ที่เครื่องอื่นจะใช้เชื่อมต่อเข้ามา

---

## 2. การเริ่มระบบให้เครื่องอื่นเข้าถึงได้

ใช้คำสั่งใหม่ที่ผมเพิ่มให้ในระบบ:

1. **สำหรับเข้าใช้งานในคลินิก:**
   เปิด Terminal แล้วรันคำสั่ง:
   ```bash
   npm run dev:host
   ```
2. **สำหรับเครื่อง iPad/คอมเครื่องอื่น:**
   เปิดบราวเซอร์แล้วพิมพ์ URL:
   `http://<เลข IP ที่จดไว้>:5173`
   เช่น `http://192.168.1.50:5173`

---

## 3. การเปิดระบบจอง LINE ให้ลูกค้าภายนอก (Public Portal)

เราจะใช้ **Cloudflare Tunnel** เพื่อเปิดเฉพาะหน้าจองคิว โดยไม่ต้องยุ่งกับเล้าเตอร์

### ขั้นตอนการตั้งค่า:
1. ดาวน์โหลดและติดตั้ง [Cloudflare Tunnel (cloudflared)](https://github.com/cloudflare/cloudflared/releases)
2. รันคำสั่งนี้เพื่อให้ลูกค้าเข้าถึงหน้าจองได้:
   ```bash
   cloudflared tunnel --url http://localhost:5173
   ```
3. Cloudflare จะให้ URL มาตัวหนึ่ง (เช่น `https://random-words.trycloudflare.com`)
4. **ลิงก์สำหรับส่งให้ลูกค้าจอง:** 
   ให้เติม `/line-portal` ต่อท้าย URL นั้น เช่น:
   `https://random-words.trycloudflare.com/line-portal`

> [!CAUTION]
> **ความปลอดภัย:** อย่าแชร์ URL หลักให้ลูกค้า (ให้แชร์แบบเติม /line-portal เท่านั้น) เพราะระบบ Admin จะปลอดภัยที่สุดเมื่อเข้าใช้งานผ่าน Wi-Fi ภายในคลินิกเอง

---

## 4. การจัดการฐานข้อมูล (Supabase Hybrid)

ข้อมูลของคุณจะถูกเก็บไว้ที่ **Supabase Cloud** ตามเดิม:
- **ข้อดี:** ข้อมูลซิงค์อัตโนมัติระหว่างเครื่องในคลินิกและลูกค้าที่จองผ่าน LINE
- **ความมั่นใจ:** ถึงแม้เครื่องแม่จะดับ ข้อมูลจองของลูกค้าก็จะยังคงอยู่ในระบบ Cloud เมื่อคุณเปิดเครื่องแม่ใหม่ ข้อมูลก็จะซิงค์กลับมาทันที
